## Overview
This project is a backend-only Online Shopping Management System developed using MySQL. It manages products, customers, orders, and order details. The database is designed using relational concepts and supports sales tracking, order analysis, and revenue calculation through SQL queries.

## Database Structure
The system consists of four main tables:

### Products
Stores product information such as name, price, available stock, and category.

### Customers
Stores customer details including name, email address, and phone number.

### Orders
Maintains customer order records with order date and total order amount.

### OrderDetails
Stores detailed information about each order, including products purchased and quantity.

## Features
- Product and inventory management  
- Customer data storage  
- Order and order detail tracking  
- Top-selling product analysis  
- Recent order tracking  
- Total sales revenue calculation

## Sample Queries Included

- Find top 5 selling products
